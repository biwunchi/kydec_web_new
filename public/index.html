01 / WHO IS THIS FOR

이런 분에게
정말 필요한 서비스입니다.

혼자라도 시작할 수 있고, 같은 직군끼리 모인 팀도 부족한 역할을 AI로 채울 수 있습니다.

🎓

IT 예비 개발자

처음 개발 공부 시작하는 분. 어떤 직군이 필요한지 모르겠다면 AI가 8개 역할을 시뮬레이션해줍니다.

💼

취업 준비생

포트폴리오에 보여줄 협업 경험이 필요한 분. 회의·결정·이슈·PR이 자동 기록되어 그대로 증거 자료가 됩니다.

💡

1인 아이디어 창업자

아이디어는 있는데 팀이 없는 분. PM·BE·FE·DS·ML·OPS·QA를 AI로 채워 6인 팀처럼 굴립니다.

📚

부트캠프 교육생

같은 직군 동기들끼리 모인 팀. 부족한 역할(예: BE반인데 디자인 필요)을 AI 페르소나로 즉시 보충.

🚀

프로젝트 추진 개발자

업무 외 사이드 프로젝트를 굴리고 싶은 현직 개발자. 부담 없이 시작, AI가 부족한 역할 커버.

🎯

캡스톤·동아리 팀

학기 중 팀원이 빠지거나 시험기간 마비되는 팀. AI가 빈자리를 메우고 회의는 계속 굴러갑니다.

01

디자이너 1명이 없어서

같은 시간대·같은 의지·같은 열정의 팀원 찾기가 진짜 어렵습니다. 디자이너 한 명만 못 구해도 전체 일정이 4주씩 밀립니다.

02

한 명만 빠져도 마비

시험기간·인턴·이직. 4~6인 팀에서 한 명만 빠져도 모든 협업이 멈춥니다. 대기 시간이 작업 시간보다 길어집니다.

03

흔적이 안 남는 협업

회의는 했지만 결정은 GitHub 이슈로 안 옮겨지고, PR 리뷰는 없고, 진척도는 단톡방 보고. 포트폴리오에 흔적이 없습니다.

04

AI 도구는 따로따로

Midjourney·v0·Gamma·Cursor. 도구마다 따로 가입·결제·학습. AI가 협업 안으로 안 들어옵니다.

03 / HOW IT WORKS

디스코드를 켜는 순간,
팀이 굴러가기 시작합니다.

평균 5분. Discord와 GitHub만 있으면 베타 시작 가능합니다.

STEP 01

프로젝트 만들기

이름·목표·기간·산출물만 정하면 필요한 역할 슬롯이 자동 생성. 사람·AI·모집 중으로 자유 지정.

STEP 02

디스코드 연결

봇 초대 후 #ai-팀-회의 채널 자동 생성. 페르소나별 webhook으로 회의 시작.

STEP 03

AI 회의 자동 진행

결정·할일·미결정 질문이 자동 추출. GitHub 이슈, Notion 페이지, 시안까지 자동 생성.

STEP 04

사람은 검토만

outcomes 카드의 ✓/✗ 클릭만. action item 완료/재오픈 토글. 미결정은 다음 회의에서 재론.
